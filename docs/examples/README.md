# Comparative Examples

This directory contains small educational examples that contrast
non-compliant and compliant code patterns inspired by the
NASA/JPL "Power of Ten" rules.

## Purpose

The goal of these examples is not to provide production-ready code,
but to make each rule easier to understand through direct comparison.

## Structure

For each rule, two files are provided:

- `*_bad.c`: a deliberately non-compliant example
- `*_good.c`: a simple compliant alternative

## Notes

Some "bad" examples are intentionally simplified.
They are included only to highlight one design issue at a time.
